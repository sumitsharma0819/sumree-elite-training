# GitHub Pages upload

Upload the contents of this folder to the root of your GitHub repository and enable GitHub Pages from `main` and `/(root)`.

## Main files
- `index.html` → couple landing page
- `sumo.html` → Sumo training system
- `ree.html` → Ree cardio + sculpt system

## Interactive landing behavior
- Hover/tap the male face → opens `sumo.html`
- Hover/tap the female face → opens `ree.html`
